# What is it?

Coder CDE is an application for cloud development environemnts. 
Users can access their own workspaces that run as full fledged machines. 
Workspaces can be accessed through the Coder portal or through IDE integrations to directly open directories remotely. 

# Why?

Coder is much more beginner friendly for those that want a reproducible development environment for projects without having to overtake the sometimes complex procress of configuring them. Project maintainers can provide dev containers which contain the exhaustive set of requirements for a project to be developed.

# How?

Coder can be accessed [here](https://coder.umlcloudcomputing.org). Authenticate with your GitHub account. 

:::note

Your GitHub account must be an accepted memeber of the UMLCloudComputingClub organization to be able to access these development environements. If you're not a member of the organization, [reach out](mailto:cloudcomputing@uml.edu).

:::
