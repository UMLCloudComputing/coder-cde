# Contributing

Workspaces are provisioned by applying terraform templates on Coder.

Continuous Development jobs as GitHub Actions automatically run when there are changes in the `/templates` directory on the repository. 

If you want to contribute with terraform templates:
1. Clone the repository
2. Make a new branch with the following name structure`feat/...`
3. Make your edits to any templates in the `/templates` directory
4. Commit your changes, publish your changes, and make a pull request
5. Make any additional changes to your work if requested by the reviewers
6. Once accepted, your changes will be merged and your template should go live on Coder.
