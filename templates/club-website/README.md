# Club Website Devcontainer Workspace

Workspace specifically for the club website. 
Packaged as a devcontainer

🍱 What's included:
- `npm` Node.js package manager
- `mkcert` CLI tool
- `npm` dependencies pre-installed

Upon first time creation, be sure to install `npm` dependencies by running `npm i` in the terminal. 
This is only needed for the first time the workspace is created. It is not necessary for subsequent interactions with your workspace.
